Tip Example
Use high-contrast, easy-to-read color schemes. The color scheme of the main content of this page (default white background with black font) as well as those of the page header (royal blue background with white font) and navigation menu (light yellow with blue link font) are intended to be easy to read. Click the button below to toggle the main content between the easy-to-ready black text with white background and a hard-to-read orange text on red background for the sake of illustration.
Offer a few different contrast options to serve those with different visual needs and preferences. This website uses a light color scheme by default. Be sure when creating other color schemes to change the color schemes of all necessary elements (in addition to text and background colors, consider things such as table borders and link colors).

See the link below for a great tool for calculating the hexadecimal inverse of a given color:
https://www.colortools.net/color_complementary.html
Use a font that is easy to read for those with processing disorders such as dyslexia. This website is written in a free, open source font called OpenDyslexic that is designed to make content easier to read for those with dyslexia. Letters are weighted and clearly differentiated from one another. This font can be used on any website without attribution.

Here is the link to the official website of OpenDyslexic: https://www.opendyslexic.org/

Here is the link to a webpage explaining how to implement OpenDyslexic on your own website:
https://github.com/antijingoist/open-dyslexic/wiki/How-to:-Use-on-your-website
